# DSA Code Quiz App

A interactive web-based quiz application focused on Data Structures and Algorithms (DSA) concepts. Test your knowledge across multiple programming languages including C++, Java, Python, and JavaScript.

## Features

- **Multiple Languages**: Questions covering DSA concepts in C++, Java, Python, and JavaScript
- **Interactive Quiz**: Multiple choice questions with immediate feedback
- **Timer**: 30-second timer for each question to keep you engaged
- **Progress Tracking**: Visual progress bar showing your advancement through the quiz
- **Scoring System**: Track your correct answers and final score
- **Responsive Design**: Works on desktop and mobile devices
- **Modern UI**: Clean, modern interface with smooth animations

## Technologies Used

- **HTML5**: Structure and layout
- **CSS3**: Styling and responsive design
- **JavaScript**: Quiz logic, timer, and interactivity

## How to Use

1. Open `index1.html` in your web browser
2. Select your preferred programming language
3. Answer the multiple-choice questions within the time limit
4. View your final score at the end

## Questions Included

- C++: 6 questions on DSA concepts
- Java: 6 questions on DSA concepts
- Python: 6 questions on DSA concepts
- JavaScript: 6 questions on DSA concepts

## Getting Started

Simply clone this repository and open `index1.html` in any modern web browser. No additional setup required!
